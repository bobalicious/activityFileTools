Settings

Everything the tools remember, in one place. It is stored in this browser on this computer — nothing is uploaded anywhere.

1What is stored

2Back up and restore

Export writes a single file covering every tool. Import reads one back — useful for moving to another browser or computer, or before clearing this one. An import replaces whatever the file contains and leaves the rest alone.

3Clear

Removes saved settings from this browser. It cannot be undone, so export first if you might want any of it back.
