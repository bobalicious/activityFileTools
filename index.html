Activity File Tools

Read, graph, correct and build FIT and GPX activity files — entirely in your browser. Nothing is uploaded anywhere.

  • Stairinator

    Turn a stair-machine workout into a FIT activity file with accurate altitude, distance, cadence and your recorded heart rate.

  • Lap Graphs

    Load a FIT file and chart it — metric rows, HR zones, rest detection, per-length swim data, and PNG export.

  • Swim FIT Corrector

    Fix pool swims where the watch missed a turn, or saw one that never happened, and write a corrected file to re-upload.

Runs from file:// — no build, no server, no network. Open index.html and go.
